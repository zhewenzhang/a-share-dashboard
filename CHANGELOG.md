## [2.1.0] - 2026-03-14

### ✨ 第二阶段：投资组合管理

#### 新功能
- **持仓管理** - 完整的股票持仓管理系统
  - 添加/删除持仓
  - 持仓数量、成本价记录
  - 实时市值计算
- **盈亏计算** - 实时盈亏追踪
  - 总盈亏统计
  - 个股盈亏计算
  - 收益率百分比
- **风险指标** - 专业投资风险分析
  - 最大回撤
  - 夏普比率
  - 波动率
  - 胜率统计
- **预警系统** - 智能投资预警
  - 跌幅超 10% 预警
  - 涨幅超 20% 止盈提醒
  - 预警历史记录
- **行业分布** - 持仓可视化
  - ECharts 饼图展示
  - 行业占比分析
- **数据导出** - 投资组合 CSV 导出

#### 数据存储
- **本地存储** - localStorage 持久化
  - 持仓数据自动保存
  - 预警记录本地存储
  - 配置数据同步

#### UI 改进
- **投资组合页面** - 全新设计的 portfolio.html
- **统计卡片** - 4 个核心指标展示
- **持仓表格** - 完整的持仓明细表格
- **风险指标网格** - 4 个风险指标卡片
- **预警列表** - 实时预警通知

### 🔧 技术实现
- 使用 localStorage 持久化存储
- 模拟价格数据（后续接入 TuShare）
- ECharts 饼图可视化
- 自动预警检查（每分钟）

## [2.0.0] - 2026-03-14

### ✨ 重大更新

#### UI 重构
- **全新现代化设计** - 采用深色主题，高级感十足的视觉效果
- **侧边栏导航** - 专业的侧边栏布局，包含主菜单、工具、系统三大分类

#### 新功能

- **数据源切换** - 支持模拟数据和 TuShare API 一键切换
- **API 状态指示** - 顶部栏显示当前数据源状态
- **Toast 通知系统** - 优雅的操作反馈提示

#### 导航系统
- **数据看板** - 实时监控页面

- **投资组合** - 持仓管理入口（开发中）
- **策略回测** - 回测功能入口（开发中）
- **模拟交易** - 模拟盘入口（开发中）
- **系统设置** - 配置管理
- **API 状态** - 服务检查

#### 交互优化
- **移动端适配** - 响应式侧边栏，支持移动端菜单
- **详情弹窗** - 改进的股票详情展示
- **导出功能** - 图表 PNG 导出和 CSV 导出
- **搜索筛选** - 股票搜索和行业筛选

### 🔧 技术改进

- 优化 CSS 变量系统
- 改进背景装饰效果

## [1.0.0] - 2026-03-14

### ✨ 新增

#### 核心功能
- **资金流向看板** - 实时展示主要指数和板块资金流向
- **智能选股推荐** - 基于资金强度评分生成推荐列表
- **板块资金排行** - 行业/概念板块资金流入排名 TOP10
- **资金强度评分** - 多维度量化资金强度

#### 后端服务
- `TuShareService` 数据服务类
- `CapitalAnalyzer` 资金分析引擎
- RESTful API 接口

#### API 接口
- `GET /api/health` - 健康检查
- `GET /api/capital-flow/{ts_code}` - 个股资金流向
- `GET /api/recommendations` - 智能推荐
- `GET /api/sectors/flow` - 板块资金流向
- `GET /api/indices` - 主要指数

#### 前端页面
- 深色金融终端主题 UI
- 指数卡片、板块图表、推荐表格
- 响应式设计
