# 行业资金流向功能说明

## 功能概述

在首页添加了**行业资金流向**展示组件，让用户可以一目了然地看到资金流入/流出哪些行业。

## 功能特性

### 1. 数据展示
- **行业排名**: 按净流入金额排序，前3名高亮显示
- **资金流向**: 绿色表示流入，红色表示流出
- **进度条**: 直观展示相对资金量
- **龙头股**: 显示各行业资金流入最多的股票
- **统计信息**: 顶部显示总流入/流出金额和行业数量

### 2. 交互功能
- **点击查看**: 点击行业项显示详细信息
- **刷新按钮**: 手动刷新行业数据
- **自动加载**: 页面加载时自动获取数据

### 3. 数据结构
```json
{
  "industry_code": "801080",
  "industry_name": "电子",
  "net_amount": 2850000000,
  "stock_count": 85,
  "up_count": 62,
  "down_count": 23,
  "avg_change_pct": 2.35,
  "leader_stock": {
    "ts_code": "688981.SH",
    "name": "中芯国际",
    "net_amount": 450000000
  }
}
```

## 技术实现

### 后端 (backend/app/services/industry_service.py)
- **IndustryFlowService**: 行业资金流向服务类
- **数据获取**: 支持从 TuShare 获取真实数据（当前使用模拟数据）
- **缓存机制**: 自动缓存到 SQLite，15分钟过期
- **行业映射**: 申万一级行业分类

### API 端点
```
GET /api/industry/flow
GET /api/industry/{industry_code}/stocks
```

### 前端 (index.html)
- **组件位置**: 首页右侧卡片（原"资金强度 TOP5"位置）
- **样式设计**: 专业的资金流向展示，支持深色主题
- **实时更新**: 自动加载，支持手动刷新

## 展示内容

### 头部统计
- 资金流入总金额
- 资金流出总金额
- 流入/流出行业数量

### 行业列表
每个行业显示：
1. 排名（前3名高亮）
2. 行业名称
3. 成分股数量
4. 涨跌家数统计
5. 龙头股名称
6. 净流入金额
7. 资金流向进度条

## 使用说明

### 查看行业资金流向
1. 打开首页 http://localhost:8080
2. 右侧"行业资金流向"卡片自动加载数据
3. 查看各行业资金流入/流出情况
4. 点击行业项查看详细信息

### 刷新数据
1. 点击卡片右上角"刷新"按钮
2. 等待数据加载完成
3. 查看最新行业资金流向
